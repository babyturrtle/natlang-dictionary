    Add phrase word - Help

    This is a page that allows the user to add a new word that can form a phrase with the given word to the dictionary or to form a connection with an existing word if it is already present in the database.

    To do that, input a word into the textarea below and click on the "Add phrase word" button.

    To go back to the main dictionary page with the list of all the words, click on the "View dictionary" button.

    You can also get to the main dictionary page by clicking the header on top of the page.

    To go back to editing the word click on the "Edit word" button.
