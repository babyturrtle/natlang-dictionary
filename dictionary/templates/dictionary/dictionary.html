    Dictionary for phrase formation - Help

    This dictionary can contain a list of words in alphabetical order.
    The words store other words that can make up phrases together.
    The user can input a text to add new words and their connections to the dictionary, as well as a single word.

    Before adding new words or editing the database the user is prompted to register or log in the system.
    To do that click the "Register" button to create a new user or the "Log In" button to if the user information already exists in the system.

    To view a word and its list of words for phrase formation, as well as some key phrases, click on the word.

    To add new words, their connections and key phrases by text, click on the "Add text" button.

    To add a single new word to the dictionary click on the "Add word" button.
